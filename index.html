Connect GitHub Storage

Your watchlist is saved as watchlist.json in your GitHub repo — synced across every device.

  1. Create a public GitHub repo (e.g. my-watchlist)
  2. Go to GitHub → Settings → Developer Settings → Fine-grained tokens
  3. Generate a token — set Repository access to your repo, enable Contents → Read & write
  4. Paste everything below and hit Save
