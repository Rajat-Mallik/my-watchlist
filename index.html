Connect GitHub Storage

Your watchlist will be saved as a watchlist.json file inside your GitHub repo — synced across every device.

  1. Create a public GitHub repo (e.g. my-watchlist)
  2. Go to GitHub → Settings → Developer Settings → Fine-grained tokens
  3. Click Generate new token — set Repository access to your repo only, and enable Contents → Read & write
  4. Copy the token and paste it below
